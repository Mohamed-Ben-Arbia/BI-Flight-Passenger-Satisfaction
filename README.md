# BI Flight Passenger Satisfaction Project

## Project Description
This project develops a full Business Intelligence (BI) solution to analyze airline passenger satisfaction data.
The objective is to identify key drivers of satisfaction, track performance KPIs, and support data-driven decision making.

## Dataset
- Source: Kaggle (Airline Passenger Satisfaction Dataset)
- Type: Public and anonymized dataset
- Size: 25,000+ rows

## Repository Structure
- data_raw: Original dataset
- data_cleaned: Cleaned dataset (ETL output)
- etl: Data cleaning and transformation scripts
- model: Star schema and measures definitions
- dashboard: Power BI dashboards
- report: Use case and insights reports
- presentation: Final project presentation
